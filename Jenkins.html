Introduction to Jenkins

What is Jenkins?

Jenkins is an opensource automation tool which is used for continuous integration. Some of below features of Jenkins
1. Jenkins can be used as a simple CI server or turned into the continuous delivery hub for any project.
2. Easy installation
3. Easy configuration
4. It provides 400+ Pulgin
5. Jenkins provides extensible features by adding plugins and distributed by build different projects and testing etc.

Prerequisites

1. RHEL 8
2. Minimum 1 GB RAM
3. JDK Higher verison is supported
4. Open Port 8080.

To install Jenkins follow these steps:

1. # sudo wget -O /etc/yum.repos.d/jenkins.repo https://pkg.jenkins.io/redhat-stable/jenkins.repo

2. # sudo rpm --import https://pkg.jenkins.io/redhat-stable/jenkins.io.key

3. # yum install jenkins

4. After installing Jenkins, to check whether it is working or not; Run Command # -systemctl status jenkins, if it is not in active state then Run Command -# systemctl start jenkins.

5. To know Ip address of your system Run Command - # ifconfig Once Jenkin is downloaded.

6. Now copy your IP address followed by port 8080, say "192.168.1xx.xxx:8080" and paste it in your browser, you can see jenkins portal which opens and ask for password. Password file location is given. Goto to location and copy the password and paste it in the password input box.

7. After this step you will be redirected to Jenkins Dashboard portal where you can start doing your tasks like creating jobs, downloading plugins, etc.

8. To install plugin goto; Manage Jenkins --> Manage Plugins --> Available --> Search for Github and select and then click on Download and install.

9. Now you are successfully completed the setup of Jenkins and installed Github plugin.
